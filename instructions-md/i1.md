# **Final Action Logic**

## **Core States**

* **Active (default)**
* **Editing**
* **Inactive (soft-deleted)**

---

# **1. View**

### **Purpose**

* Read-only access

### **Behavior**

* Open details (modal / page / drawer)
* No state change

### **Availability**

* ✅ Active
* ❌ Editing
* ❌ Inactive *(based on your table — hidden there)*

---

# **2. Edit ↔ Cancel (Toggle Pair)**

## **Edit**

### **Purpose**

* Enter edit mode

### **Behavior**

* Switch row → **editing state**
* Replace editable fields with inputs (atomic)
* Store original data snapshot

### **Availability**

* ✅ Active
* ❌ Editing
* ❌ Inactive

---

## **Cancel**

### **Purpose**

* Exit edit mode

### **Behavior**

* Revert all changes to original snapshot
* Exit edit mode → back to **Active**

### **Availability**

* ❌ Active
* ✅ Editing
* ❌ Inactive

---

# **3. Deactivate ↔ Restore (Soft Delete Toggle)**

## **Deactivate**

### **Purpose**

* Soft delete (disable record)

### **Behavior**

* Set:

```js
isActive = false
```

* Move row → **Inactive state**
* Do NOT remove from DB

### **Availability**

* ✅ Active
* ❌ Editing
* ❌ Inactive

---

## **Restore**

### **Purpose**

* Undo soft delete

### **Behavior**

* Set:

```js
isActive = true
```

* Move row → **Active state**

### **Availability**

* ❌ Active
* ❌ Editing
* ✅ Inactive

---

# **4. Delete (Hard Delete)**

### **Purpose**

* Permanently remove record

### **Behavior**

* Require confirmation. **In the confirmation modal/dialog, list where this record is being used or used by**.
* Call delete API
* Remove from UI list

### **Availability**

* ✅ Active
* ❌ Editing
* ✅ Inactive *(allowed for cleanup)*

---

# **5. State → Button Visibility Matrix**

| State    | Visible Buttons                |
| -------- | ------------------------------ |
| Active   | View, Edit, Deactivate, Delete |
| Editing  | Cancel                         |
| Inactive | Restore, Delete                |

---

# **6. Critical Behavior Rules**

### **Mutual Exclusivity**

* Edit ❌ Cancel (never together)
* Deactivate ❌ Restore (never together)
