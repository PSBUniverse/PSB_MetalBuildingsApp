# 🧱 1. FAKE PANEL LINES (NO GEOMETRY SPAM)

👉 Stop using boxes/lines for panels. That’s why your model exploded.

## ✅ Correct Approach: Use TEXTURES

You fake detail using UVs, not meshes.

---

## 🔧 Option A — Canvas Texture (Best Control)

You generate panel lines dynamically.

### Step 1: Create texture

```js
function createPanelTexture({
  width = 512,
  height = 512,
  lineSpacing = 20,
  lineWidth = 2,
  lineColor = "#222",
  bgColor = "#d32f2f"
}) {
  const canvas = document.createElement("canvas")
  canvas.width = width
  canvas.height = height

  const ctx = canvas.getContext("2d")

  // background
  ctx.fillStyle = bgColor
  ctx.fillRect(0, 0, width, height)

  // vertical panel lines
  ctx.strokeStyle = lineColor
  ctx.lineWidth = lineWidth

  for (let x = 0; x < width; x += lineSpacing) {
    ctx.beginPath()
    ctx.moveTo(x, 0)
    ctx.lineTo(x, height)
    ctx.stroke()
  }

  const texture = new THREE.CanvasTexture(canvas)
  texture.wrapS = THREE.RepeatWrapping
  texture.wrapT = THREE.RepeatWrapping

  return texture
}
```

---

## Step 2: Apply to roof

```js
const texture = createPanelTexture({
  bgColor: "#b71c1c"
})

texture.repeat.set(10, 2) // controls panel density

const material = new THREE.MeshBasicMaterial({
  map: texture
})
```

# 🧱 2. MAKE IT LOOK LIKE METAL (WITHOUT HEAVY LIGHTING)

You’re using `MeshBasicMaterial`, so:

👉 No lighting = no realism
👉 We fake realism with **color variation + shading tricks**

---

## 🔧 Option A — Fake Lighting Gradient (IMPORTANT)

Modify canvas:

```js
const gradient = ctx.createLinearGradient(0, 0, 0, height)
gradient.addColorStop(0, "#ff4d4d")
gradient.addColorStop(1, "#7f0000")

ctx.fillStyle = gradient
ctx.fillRect(0, 0, width, height)
```

👉 Now roof has depth without lights

---

## 🔧 Option B — Subtle Noise (Break flat look)

```js
for (let i = 0; i < 2000; i++) {
  const x = Math.random() * width
  const y = Math.random() * height

  ctx.fillStyle = "rgba(0,0,0,0.03)"
  ctx.fillRect(x, y, 1, 1)
}
```

👉 Removes “perfect plastic” look

# 🧱 3. CORRECT PANEL DIRECTION (THIS MATTERS)

Right now most devs mess this up.

👉 Roof panels should run along **length**

So:

```js
texture.repeat.set(length / 5, 1)
```

👉 Not width.
